# Homebrew Tap

Homebrew formulae and casks for JaysonRawlins projects.

## Installation

```bash
brew tap jaysonrawlins/tap
```

## Available Casks

| Cask | Description |
|------|-------------|
| `pingplace` | Control notification position on macOS |

### PingPlace

```bash
brew install --cask jaysonrawlins/tap/pingplace --no-quarantine
```
